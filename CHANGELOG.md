# CHANGELOG

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-22

### Added
- Initial release of NovaVestv1
- Telegram Bot with complete menu system
- React Mini App with responsive design
- User authentication via Telegram
- Wallet system with multiple balance types
- Investment plans and management
- Deposit and withdrawal workflows
- Referral system with tracking
- Task system with rewards
- Admin dashboard for management
- Notifications system
- Complete API documentation
- Database schema and migrations
- Security guidelines and best practices
- Deployment guides for Render, Docker, VPS, and Heroku
- Docker Compose for local development
- Comprehensive logging and error handling
- Rate limiting and input validation

### Backend
- Express.js API server
- PostgreSQL database
- Telegraf bot framework
- JWT authentication
- Pino logging

### Frontend
- React 18
- Redux Toolkit state management
- Tailwind CSS styling
- Framer Motion animations
- Responsive design
